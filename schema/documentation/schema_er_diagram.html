🔗 Key Relationships

Projects → Blocks/Items/Enchantments: 1-to-Many relationship. Har bir loyiha ko'plab bloklarni, itemlarni va muito'chiqlarni o'z ichiga oladi.
Recipes → Ingredients: Many-to-Many orqali amalga oshirilgan. Har bir retsept ko'plab ingredientlarni o'z ichiga oladi.
Armor/Tool Types → Items: Each armor set har 4 ta item (helmet, chestplate, leggings, boots) bilan bog'langan.
Entity Types → Drops: 1-to-Many relationship. Har bir mob ko'plab tarqatmalarni (loot) o'z ichiga oladi.
Projects → Agent Tasks: Build jarayoni va code generation agentlarini kuzatydi.
Projects → Build Logs: Har qurish urinishi uchun log yaratiladi va xatolar qayd etiladi.
