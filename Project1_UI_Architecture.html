🎮 MINECRAFT MOD GENERATOR

User Interface (UI) Architecture Dokumentatsiyasi

Rasmiy Texnik Hujjat | v1.0

📋 1. KIRISH

Ushbu hujjat Minecraft Mod Generator ilovasining foydalanuvchi interfeysi (UI) arxitekturasini batafsil tavsiflaydi. UI Qt/C++ yoki Electron asosida qurilgan bo'lib, foydalanuvchiga intuytiv va operatsion tajriba taqdim etadi.

🛠️ Texnologiya Tanlovi:
  • Framework: Qt 6.x (C++) yoki Electron (JavaScript)
  • Backend Integration: REST API / gRPC
  • Styling: Material Design / Fluent Design
  • Database Connection: SQLite (local)

⚙️ 5. FOYDALANUVCHI VORKFLOU
📍 Bosqich 1: Proyekt Yaratish
  • Dashboard'da "New Project" tugmasini bosing
  • Dialog oynasi ochiladi
  • Proyekt nomini, Minecraft versiyasini va Mod Loader'ni tanlang
  • "Create" tugmasini bosing
  • Workspace oynasi ochiladi
📍 Bosqich 2: Blok Yaratish
  • Workspace'da "Blocks" papkasini tanlang
  • Right-click → "New Block"
  • Dialog oynasida blok xususiyatlarini kiriting
  • Hardness va Resistance'ni slider orqali sozlang
  • "Create" tugmasini bosing
  • Blok File Explorer'da ko'rinadi
📍 Bosqich 3: Qurish va Sinov
  • ToolBar'da "Build" tugmasini bosing
  • Gradle avtomatik kompilyatsiya qiladi
  • Console'da build loguylari ko'rsatiladi
  • Build muvaffaq bo'lsa, "Run" tugmasini bosing
  • Minecraft Sandbox o'chadi va mod jonli rejimda tekshiriladi

♿ 9. ACCESSIBILITY (A11Y)
  • Keyboard Navigation: Barcha elementlar Tab tugmasi bilan navigatsiya qilinishi kerak
  • Screen Reader Support: QT's accessibility framework ishlatish
  • Font Sizing: Foydalanuvchi shrift o'lchamini o'zgartiradish mumkin
  • Color Contrast: WCAG AA standartiga mos rangli contrast
  • High Contrast Mode: Yanada yaxshi ko'rinish uchun
  • Tooltip va Help Text: Hover'da qisqa tavsif ko'rsatish
⚠️ 10. ERROR HANDLING VA USER FEEDBACK

Error Turlari:

Error Turi Foydalanuvchiga Ko'rsatilishi Misol
Validation Error Dialog'da qizil rang bilan warning "Blok nomini kiriting"
File Error Toast notification (pastki burchakda) "Fayl saqlashda xato"
Build Error Console'da red rang bilan xato "Gradle compilation failed"
Network Error Retry dialog bilan modal popup "AI agentga ulanishda xato"
